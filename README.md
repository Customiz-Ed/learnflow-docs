# CustomizEd Docs

Frontend application for CustomizEd built with React, TypeScript, Vite, and Tailwind CSS.

## Getting started

Install dependencies:

```sh
bun install
```

Start the development server:

```sh
bun run dev
```

Build for production:

```sh
bun run build
```

Run tests:

```sh
bun run test
```

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
